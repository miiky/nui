输入框

模块继承 placeholder 模块，包含其所有功能，增加清除输入框内容按钮以及密码框显示明文按钮，也可自定义按钮

案例

选项参数

# iconfont

默认值：false

类型：{String} 设置字体图标类名，会在按钮上添加上该类

类型：{Boolean} 值为true是类名默认是iconfont

# hover

默认值：false

类型：{Boolean} 在输入框有值的情况下，是否鼠标悬停在输入框上时再显示按钮

# show

默认值：false

类型：{Boolean} 是否一直显示按钮，当输入框值为空时也显示

# reveal

默认值：null

类型：{String} 设置密码框显示隐藏按钮内容，配合iconfont参数值可设置为字体图标编码

类型：{Object}

属性 默认值 类型 说明
content null {String} 设置按钮内容
{Object}
属性 默认值 类型 说明
text null {String} 当输入框为text类型时按钮内容
password null {String} 当输入框为password类型时按钮内容
title null {String} 设置按钮提示文本
{Boolean} 当值为true时，去按钮内容作为提示文本，false则不启用提示功能
{Object}
属性 默认值 类型 说明
text null {String} 当输入框为text类型时提示文本
password null {String} 当输入框为password类型时提示文本
iconfont null {Boolean} 功能等同选项参数
hover null {Boolean} 功能等同选项参数
show null {Boolean} 功能等同选项参数
callback null {Function} 按钮触发回调，接收3个参数依次是self:实例对象，event:事件对象，elem:按钮jQuery对象

# clear

默认值：null

类型：{String} 设置清除内容按钮内容，配合iconfont参数值可设置为字体图标编码

类型：{Object}

属性 默认值 类型 说明
content null {String} 设置按钮内容
title null {String} 设置按钮提示文本
{Boolean} 当值为true时，去按钮内容作为提示文本，false则不启用提示功能
iconfont null {Boolean} 功能等同选项参数
hover null {Boolean} 功能等同选项参数
show null {Boolean} 功能等同选项参数
callback null {Function} 按钮触发回调，接收3个参数依次是self:实例对象，event:事件对象，elem:按钮jQuery对象

# button

默认值：null

类型：{Array} 设置多按钮，参数等同clear配置

# text

默认值：''

类型：{String} 输入框占位提示文本，若元素上含有placeholder属性将会覆盖该值

# animate

默认值：false

类型：{Boolean} 是否使用动画形式显示占位文本

# equal

默认值：true

类型：{Boolean} 输入框值是否可以和占位符相同，设置为false时如果输入值等于占位符值将自动被清空

# restore

默认值：true

类型：{Boolean} 销毁或者重置组件是否还原默认值

# color

默认值：'#ccc'

类型：{String} 占位符文本颜色

# id

默认值：''

类型：{String} 标识符名称，弹出层最外层元素会增加layer-标识符名称的类名

# skin

默认值：''

类型：{String} 皮肤名称，弹出层最外层元素会增加nui-layer-皮肤名称的类名

# onChange

默认值：null

类型：{Function} 调用实例的value方法后执行回调

实例属性与方法

# value

类型：{Function}

参数：val {String}

说明：设置输入框值

# option

类型：{Function}

参数：option {String, Object} 选项参数集合或者参数名

参数：value {Anything} 参数值，option为string类型时才会使用

说明：设置选项参数

# reset

类型：{Function}

说明：将实例还原为初始状态

# destroy

类型：{Function}

说明：销毁实例

静态方法

# config

类型：{Function}

参数：option {String, Object} 选项参数集合或者参数名

参数：value {Anything} 参数值，option为string类型时才会使用

说明：全局设置默认选项参数，需在调用组件前设置方可生效

# value

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：将实例还原为初始状态，不传id则对所有实例操作

# option

类型：{Function}

参数：option {String, Object} 选项参数集合或者参数名

参数：value {Anything} 参数值，option为string类型时才会使用

参数：id {Number, String} 实例id或者选项参数id

说明：设置选项参数，不传id则对所有实例操作

# reset

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：将实例还原为初始状态，不传id则对所有实例操作

# resize

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：改变实例大小位置为正常状态，不传id则对所有实例操作

# destroy

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：销毁实例，不传id则对所有实例操作
