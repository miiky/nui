# option

类型：{Function}

参数：option {String, Object} 选项参数集合或者参数名

参数：value {Anything} 参数值，option为string类型时才会使用

说明：设置选项参数

类型：{Function}

说明：将实例还原为初始状态

# destroy

类型：{Function}

说明：销毁实例
