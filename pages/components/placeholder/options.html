# text

默认值：''

类型：{String} 输入框占位提示文本，若元素上含有placeholder属性将会覆盖该值

# animate

默认值：false

类型：{Boolean} 是否使用动画形式显示占位文本

# equal

默认值：true

类型：{Boolean} 输入框值是否可以和占位符相同，设置为false时如果输入值等于占位符值将自动被清空

# restore

默认值：true

类型：{Boolean} 销毁或者重置组件是否还原默认值

# color

默认值：'#ccc'

类型：{String} 占位符文本颜色

# id

默认值：''

类型：{String} 标识符名称，弹出层最外层元素会增加layer-标识符名称的类名

# skin

默认值：''

类型：{String} 皮肤名称，弹出层最外层元素会增加nui-layer-皮肤名称的类名

# onChange

默认值：null

类型：{Function} 调用实例的value方法后执行回调
