占位符

使用模块

下载 最新组件作为依赖项引入到模块中，并创建实例。

选项参数

# text

类型：{String}

默认值：''

说明：输入框占位提示文本，若元素上含有placeholder属性将会覆盖该值

# animate

类型：{Boolean}

默认值：false

说明：是否使用动画形式显示占位文本

# equal

类型：{Boolean}

默认值：false

说明：输入框值是否可以和占位符相同

# restore

类型：{Boolean}

默认值：true

说明：销毁或者重置组件是否还原默认值

# color

类型：{String}

默认值：'#ccc'

说明：占位符文本颜色

# id

类型：{String}

默认值：''

说明：标识符名称，弹出层最外层元素会增加layer-标识符名称的类名，

# skin

类型：{String}

默认值：''

说明：皮肤名称，弹出层最外层元素会增加nui-layer-皮肤名称的类名

# onChange

类型：{Function}

默认值：null

说明：调用实例的value方法后执行回调

实例属性与方法

# value

类型：{Function}

参数：val {String}

说明：设置输入框值

# option

类型：{Function}

参数：option {String, Object} 选项参数集合或者参数名

参数：value {Anything} 参数值，option为string类型时才会使用

说明：设置选项参数

# reset

类型：{Function}

说明：将实例还原为初始状态

# destroy

类型：{Function}

说明：销毁实例

静态方法

# config

类型：{Function}

参数：option {String, Object} 选项参数集合或者参数名

参数：value {Anything} 参数值，option为string类型时才会使用

说明：全局设置默认选项参数，需在调用组件前设置方可生效

# value

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：将实例还原为初始状态，不传id则对所有实例操作

# option

类型：{Function}

参数：option {String, Object} 选项参数集合或者参数名

参数：value {Anything} 参数值，option为string类型时才会使用

参数：id {Number, String} 实例id或者选项参数id

说明：设置选项参数，不传id则对所有实例操作

# reset

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：将实例还原为初始状态，不传id则对所有实例操作

# resize

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：改变实例大小位置为正常状态，不传id则对所有实例操作

# destroy

类型：{Function}

参数：id {Number, String} 实例id或者选项参数id

说明：销毁实例，不传id则对所有实例操作
